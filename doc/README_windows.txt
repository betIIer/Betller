Betller Core
=====================

Intro
-----
Betller coin - secure and decentralized open-source cryptocurrency with instant and anonymous transactions.


Setup
-----
Unpack the files into a directory and run betller-qt.exe.

Betller Core is the original Betller client and it builds the backbone of the network.
However, it downloads and stores the entire history of Betller transactions;
depending on the speed of your computer and network connection, the synchronization
process can take anywhere from a few hours to a day or more.
